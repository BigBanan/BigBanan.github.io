利用 nginx 反向代理

当只有一台服务器，而又想运行多个 web 后端服务，且希望网址为 www.XX.com / bbs.XX.com / blog.XX.com,怎么办，此时就要用nginx进行反向代理

以 ubuntu 为例

1. 安装 nginx

2. 修改 nginx 的配置文件

然后修改 nginx.conf,
具体为:修改配置文件中 的 http{}
在 http{} 中的末尾补上反向代理配置代码

3. 在重启 nginx 服务

4. 在服务器启动 blog、bbs、www 服务

注意：
端口号 8081 为 bbs 服务, 8082 为 blog 服务,8083 为 www 服务
