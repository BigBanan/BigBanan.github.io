this & arguments

2016-12-26

在函数内部,有两个特殊的对象: arguments 和 this

1. arguments 主要用途是 保存函数参数

argument 对象还要一个 callee 的属性,该属性指向 拥有 arguments 对象的函数

2. this 对象代表函数据以执行的执行环境的变量对象(可参考上一文章)

另ES5 给函数添加了 caller 属性(作用:指向调用当前函数的函数(即当前函数他爸))
